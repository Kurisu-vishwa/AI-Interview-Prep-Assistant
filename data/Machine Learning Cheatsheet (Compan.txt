2. Essential Formulas
Linear Regression
Prediction: y = b0 + b1x
Cost (MSE): J = (1/n) Σ(y - y_hat)^2
Logistic Regression
Sigmoid: P(y=1)=1/(1+e^-z)
Log loss: -1/n Σ[y log(p)+(1-y)log(1-p)]
Gradient Descent
theta = theta - alpha * gradient
alpha = learning rate
Bayes Theorem
P(A|B)= P(B|A)P(A)/P(B)

3. Evaluation Metrics
Classification
Accuracy: (TP+TN)/(TP+TN+FP+FN)
Precision: TP/(TP+FP)
Recall: TP/(TP+FN)
F1: 2PR/(P+R)
Use:
• Precision when false positives costly
• Recall when false negatives costly
Regression
MAE: mean absolute error
MSE: mean squared error
RMSE: sqrt(MSE)
R²: Explained variance
4. Bias-Variance Summary
High Bias:
• Underfitting
• Model too simple
High Variance:
• Overfitting
• Model too complex
Goal: Balance both.
2
5. Algorithm Quick Comparison
Linear Regression
Use: Simple continuous prediction Pros: Fast, interpretable Cons: Limited nonlinear power
Logistic Regression
Use: Classification baseline Pros: Simple, probabilistic Cons: Linear boundary
Decision Tree
Use: Interpretable nonlinear modeling Pros: Explainable Cons: Overfits easily
Random Forest
Use: Strong default tabular model Pros: Robust Cons: Less interpretable
XGBoost
Use: High-performance structured data Pros: Often top performer Cons: Tuning complexity
SVM
Use: High-dimensional data Pros: Strong margin classifier Cons: Scaling difficult
KNN
Use: Simple similarity tasks Pros: Easy Cons: Slow at scale
6. Bagging vs Boosting
Bagging: Parallel Variance reduction Example: Random Forest
Boosting: Sequential Bias reduction Example: XGBoost
